# Log a new decision in DECISIONS.md

Read DECISIONS.md to find the next available decision number.

Then ask me:
1. What decision was made? (one sentence)
2. What problem prompted it?
3. What alternatives were considered?
4. Is this Active, Superseded, or Deferred?

If Superseded: which decision number does it replace?

Then write the new entry in DECISIONS.md using the standard template.
If Superseded: also update the old entry's Status field.

Do not change any other file.
